#1 Display

Definisi Display: "Setiap tag pada HTML memiliki nilai default untuk properti display. Tetapi kita juga dapat mengubah perilaku dari tag tersebut dengan mengganti value-nya."

Source: https://developer.mozilla.org/en-US/docs/Web/CSS/display/

Value Display :

  • inline
  • inline-block
  • block
  • none

inline :

  1. Element HTML yang secara default tidak menambahkan garis baru ketika dibuat.
  2. Lebar dan tinggi elementnya sebesar konten yang ada didalam-nya.
  3. Kita tidak dapat mengatur tinggi dan lebar dari element inline.
  4. Margin dan padding hanya mempengaruhi element secara horizontal, tidak vertikal.

inline-block :

  1. Tidak ada element yang secara default memiliki properti display: inline-block;.
  2. Kita harus ubah secara manual properti tersebut.
  3. Perilaku dasarnya sama dengan element inline.
  4. Perbedaannya, element inline-block dapat kita atur tinggi dan lebar-nya.

Block :

  1. Element HTML yang secara dafault menambahkan garis baru ketika dibuat
  2. Jika tidaj tinggi dan lebar-nya, maka tinggi dan lebar default akan dari element block akan memenuh tinggi dan dan lebar
  3. Kita dapet mengatur tinggi dan lebar dari element block
  4. didalam element block, kita dapat menyimpan element tag dengan element inline,inline-blok, atau bahkan element inline-blok

None :

  • Digunakan untuk menghilangkan sebuah element

#4 Box Model

Definisi box model :

  • Setiap element di halaman berada didalam sebuah box(kotak)
  • Kita bisa mengatur ukuran dan posisi kotak tersebut
  • kita bisa memberi warna/gambar sebagai background kotak tersebut

Source :https://css-tricks.com/the-css-box-model/

CSS box model mendefinisikan 'kotak' yang dihasilkan oleh sebuah element, lalu menampilkannya sesuai dengan format visualnya. (Source :https://www.w3.org/TR/CSS2/box.html)

CSS box model terdiri dari margin, border, padding, dan content. (Source :https://www.w3schools.com/css/css_boxmodel.asp )

CSS Box Model
Margin Area transparant disekitar kotak(diluar border)
Padding Area transparant didalam kotak(antara content dan border)
Border Batas disekeliling content dan border
Content content sebenarnya didalam box, bisa berupa teks atau gambar
